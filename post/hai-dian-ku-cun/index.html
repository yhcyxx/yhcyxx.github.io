2023-05-19

1.海典批量库存变动接口

海典批量库存变动信息同步

1.1 请求描述

  1. 请求方式: POST
  2. 请求URL :mh/kapi/app/rm_se_b2c_oms/UpdateInventory
  3. 编码格式: application/json
  4. 是否需要登录: 是

1.2 请求参数说明

1.2.1 请求参数

参数名称 参数类型 是否必填 备注
batch object[] 是 批量库存数据
outer_iid string 是 海典商品编码
sku_id string 否 商品规格ID
warehouseId string 是 门店编码
quantity string 是 库存数量

1.3 请求示例

{
    "batch":[
        {
           	"outer_iid": "00000121",
  			"sku_id": "1231",
    		"warehouseId": "667",
   		 	"quantity": "34"
        },
         {
           	"outer_iid": "00000077",
  			"sku_id": "1412",
    		"warehouseId": "001",
   		 	"quantity": "100"
        }
    ]
}

1.4 返回参数说明

参数名称 参数类型 参数说明 备注
success boolean 是否成功
msg string 错误信息
data object 失败信息
code string 状态码 0成功1部分成功2失败

1.4.1 Data 字段说明

参数名称 参数类型 参数说明 备注
fail int 是 失败数量
faillist object[] 是 失败集合
outer_iid string 是 海典商品编码
sku_id string 否 商品规格ID
warehouseId string 是 门店编码
resMsg string 是 失败原因

1.5 返回结果示例

//失败
{
	"data": {
        "fail":1,
        "faillist":[
            {
                "outer_iid": "M1234667",
  				"sku_id": "561234",
    			"warehouseId": "667",
   		 		"resMsg": "第三方仓库匹配失败"
            }
        ]
    },
  "msg": "部分失败",
  "code": "1",
  "success": false
}
//成功
{	
	"message": "库存变动成功",
    "code": "0",
    "success": true
}
